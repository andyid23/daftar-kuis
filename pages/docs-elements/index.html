explode-quiz

Elemen kuis interaktif dengan efek confetti saat jawaban benar. Dibangun dengan LitElement + DDD Design System.

Tag

<explode-quiz></explode-quiz>

Atribut

AtributTipeDeskripsi
questionsArray (JSON)Daftar soal kustom: {question, choices, correctIndex}

Contoh Soal Kustom

<explode-quiz questions='[{"question":"Apa ibu kota Indonesia?","choices":["Bandung","Surabaya","Jakarta","Medan"],"correctIndex":2}]'></explode-quiz>

Fitur

  • Soal pilihan ganda
  • Animasi confetti jawaban benar
  • Navigasi antar soal
  • Skor akhir
  • DDD Design System + i18n

todo-list

Elemen daftar tugas interaktif dengan penyimpanan localStorage.

Tag

<todo-list></todo-list>

Fitur

  • Tambah, edit, hapus tugas
  • Centang selesai
  • Simpan otomatis ke localStorage
  • DDD Design System

attendance-system

Sistem absensi dengan logging aktivitas, heatmap, dan konfigurasi threshold.

Tag

<attendance-system></attendance-system>

Fitur

  • Pencatatan aktivitas mahasiswa
  • Heatmap aktivitas
  • Konfigurasi threshold kehadiran
  • Panel dosen / diagnostik
  • Penyimpanan localStorage

quiz-dashboard-lite

Dashboard lengkap menggabungkan kuis, absensi, heatmap, dan transparansi nilai dalam satu halaman. Menggunakan material-web untuk UI.

Tag

<quiz-dashboard-lite></quiz-dashboard-lite>

Fitur

  • Tab: Kuis, Absensi, Heatmap, Nilai
  • Integrasi Google Sheets
  • Material Design 3 (MD3)
  • Penyimpanan sheet ID di localStorage

notif-element

Elemen notifikasi toast ringan dengan berbagai tipe dan auto-dismiss.

Tag

<notif-element message="Pesan" type="success" duration="3000" visible></notif-element>

Atribut

AtributTipeDeskripsi
messageStringTeks notifikasi
typeStringinfo, success, warning, error
durationNumberDurasi ms (0 = tetap)
visibleBooleanTampilkan / sembunyikan

Metode

  • show() — Tampilkan notifikasi
  • hide() — Sembunyikan notifikasi

Event

EventDeskripsi
notif-showSaat notifikasi tampil
notif-hideSaat notifikasi sembunyi
